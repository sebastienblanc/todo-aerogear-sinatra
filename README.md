# Basic to-do app with AeroGear + Sinatra

## Usage instructions

1. Grab the sources

        git clone git@github.com:abstractj/todo-aerogear-sinatra.git
        
2. Run the migrations
        
        rake db:migrate && rake db:setup
        
3. Start the app

        ruby app.rb
        
4. Open the application at http://localhost:4567
